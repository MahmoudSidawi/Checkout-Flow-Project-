# Checkout Flow Project

This small project shows a multi-step checkout flow.
It uses `index.html` for the page, `style.css` for the styling, and `script.js` for navigation, validation, and payment simulation.

## What this project is

- A checkout demo with 5 steps: summary, personal info, address, payment, and confirmation.
- Validates form fields before allowing the user to continue.
- Simulates a payment gateway with a processing status screen.

## How to start

1. Open the `Checkout-Flow-Project-` folder in your code editor.
2. Open `index.html` in your browser.
3. If you use Live Server, right-click `index.html` and select "Open with Live Server".
4. Click through the checkout steps to test the flow.

## Files included

- `index.html` — page layout and checkout steps.
- `style.css` — colors, layout, buttons, and form styling.
- `script.js` — step handling, input validation, review summary, and simulated payment.
